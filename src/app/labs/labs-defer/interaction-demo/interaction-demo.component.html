Демонстрация триггера defer (on timer)

Контент загружается после 3 секунд после рендеринга.

@defer (on timer(3000)) { }

Демонстрация триггера defer (on immediate)

Контент загружается немедленно после рендеринга.

@defer (on immediate) { }

Демонстрация триггера defer (on interaction)

@defer (on interaction(btnLoading)) { } @placeholder (minimum 500ms) {

Нажмите кнопку для загрузки тяжелого контента.

}

Демонстрация триггера defer (on hover)

@defer (on hover(btnLoadingHover)) { } @placeholder {

Наведи на кнопку для загрузки тяжелого контента.

}
