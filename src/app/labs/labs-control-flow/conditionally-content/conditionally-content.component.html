Пример использования

@if (username) {

Привет, {{ username }}!

}

Пример использования {{ '@if с @else if и @else' }}

@if (status === 'loading') {

Загрузка данных...

} @else if (status === 'error') {

Произошла ошибка. Попробуйте снова.

} @else {

Данные успешно загружены!

}

Пример использования с сохранением переменной

@if (user?.profile; as profile) {

Имя: {{ profile.name }}

Email: {{ profile.email }}

} @else {

Профиль пользователя не найден.

}
